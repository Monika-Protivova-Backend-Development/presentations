Syllabus

Week 1: Basics

  • Lesson 1: Introduction to Backend Development - Understanding what backend is, why applications need it, and types of applications that use backends
  • Lesson 2: Kotlin Basics - Core Kotlin syntax, data types, functions, and language fundamentals
  • Lesson 3: Collections - Working with lists, maps, sets, and collection operations in Kotlin
  • Lesson 3: Testing - Introduction to testing principles and writing tests in Kotlin
  • Lesson 4: Application - Building a basic Kotlin application structure

Week 2: Kotlin SDK

  • Lesson 5: Application Routing - Creating REST APIs, implementing controllers, and testing API endpoints
  • Lesson 6: Handling Business Logic - Implementing service layer components and business logic in Kotlin applications
  • Lesson 7: Data Layer - Working with data persistence, repositories, and database interactions
  • Lesson 8: Polymorphism & Inheritance - Understanding and implementing object-oriented programming concepts in Kotlin

Week 3: Application

  • Lesson 9: The SOLID Principle - Learning object-oriented design patterns, functional programming principles, and generics
  • Lesson 10: Memory and Concurrency - Understanding memory management and concurrent programming in Kotlin
  • Lesson 11: Deployment and Operation - Deploying and operating Kotlin applications in production environments
  • Lesson 12: Final Lesson - Course review and final project preparation

What will we learn in this course?

In the first week, we will establish a solid foundation in Java programming language.

  1. Kotlin language syntax
  2. Basic programming principles in Kotlin
  3. How to test your code

During the second week, we will look at some standard Kotlin libraries to boost our productivity.

  1. Some of Kotlin's standard libraries
  2. Common design patterns and in Kotlin
  3. You will understand the SOLID principle

During the third week, we learn to how to write a Kotlin application backend with REST API

  1. MVC using Spring
  2. You will implement your own application backend with REST API
